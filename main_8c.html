Detailed Description

start point of the code for the mars rover for the 288 final project, containing all the initialization and code logic. This file is in charge of the basic movement logic behind the roomba and coordination the messages sent from a TCP connection to the wiFi board on the actual robot. It has defined the setup of all the peripherals tht will be used to interact with the environment and also the sensitivity and setup for the sensors.
