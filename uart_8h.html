Detailed Description

Header that includes methods to take bytes of data and transfer them using the uart protocol. Using 8 data bits, one stop and one start bit, the function uses interrupts to notify of commands that get recieved and sends information from peripherals like sensors back to the connected computer via a TCP connection. This doesnt block the ability of the CPU to carry on with more processes.
