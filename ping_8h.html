Detailed Description

File to use and initialize the Ping))) sensor. Defines the methods for both initializing the Ping))) sensor mounted on top of the servo and to set up the timer 3b of the microcontroller to rend correctly timed signals to the peripheran and obtain correct output readings from it, interprets the information of the sound coming back and forth and deduces using the time difference of the sound wave to determine the distance of an object.
